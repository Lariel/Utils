## Métodos disponíveis:
### GET:
#### /cpf/generate
```
Gera um CPF válido.
```
#### /cpf/generate/:x
```
Gera X CPFs válidos com base na quantidade passada por parâmetro.
```
#### /cpf/validate/:cpf
```
Retorna o status (boolean) da validação do DV de um CPF passado por parâmetro.
```
